# Weather App

Simple weather app built with Flask and OpenWeatherMap API.

## Features

- Real-time weather data
- Temperature, humidity, wind speed
- Search any city in English
- Country autocomplete (40+ countries)
- Dynamic city photo background via Unsplash API
- Always online — monitored by UptimeRobot

## Tech stack

- Python / Flask — backend
- HTML + CSS + Jinja2 — frontend
- OpenWeatherMap API — weather data
- Unsplash API — city background photos
- Docker — containerization
- Render.com — deployment (auto-deploy on git push)
- UptimeRobot — uptime monitoring
